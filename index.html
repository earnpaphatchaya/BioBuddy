3 ระยะ จากแปลงเกษตร
ถึงโรงไฟฟ้า

ระบบแบ่งการทำงานออกเป็น 3 ระยะที่ชัดเจน เกษตรกรไม่ต้องมีความรู้เทคโนโลยี แค่มี LINE ก็เริ่มได้ทันที

📱
ขั้นที่ 1 — INIT

เพิ่มเพื่อนและลงทะเบียน

เพิ่ม BioBuddy ใน LINE → Bot ส่ง Welcome Message และเชิญลงทะเบียน → กรอกชื่อ เบอร์โทร แล้วส่ง Location พิกัดแปลง ระบบตรวจสอบข้อมูลอัตโนมัติ

🗺️
ขั้นที่ 2 — MATCH

ระบบจับคู่และวางแผนขนส่ง

GIS วิเคราะห์พิกัดแปลงจากดาวเทียม → Clustering จัดกลุ่มแปลงใกล้เคียง → Route Optimization คำนวณเส้นทางรถบรรทุกที่ประหยัดที่สุด ลดต้นทุนได้ >30%

💰
ขั้นที่ 3 — EARN

รถมารับ โอนเงิน รับเครดิต

รถบรรทุกวิ่งตามเส้นทางที่คำนวณ → ชีวมวลเข้าโรงไฟฟ้า → เกษตรกรได้รับเงินโอนทันที พร้อม Carbon Credit สะสมรายเดือน

☀️
4 ฟังก์ชันหลัก

ทุกอย่างอยู่ในเมนูเดียว
บน Rich Menu

Rich Menu 4 ช่องช่วยให้เกษตรกรเข้าถึงทุกฟีเจอร์ได้ในแตะเดียว ไม่ต้องจำคำสั่ง

🌾

ลงชีวมวล (LIST)
